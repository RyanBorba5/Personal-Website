For some reason a long time ago I wanted to display the duplicates of a list that was in a txt file. I think it was either originally for a leetcode problem or maybe just because I was interested in knowing how many duplicated Elden Ring bosses there were. Regardless, this was good practice for messing around with filereaders/Filestream, streamwriter, whatever they're called. I've never been the biggest fan of reading from files, but I do know that it's an insanely useful skill. Maybe I could've made this project compatible with data that was more messy but perhaps I'll get to that someday.

                                    StreamWriter myWriter = new StreamWriter(fs);

                                    List Uniques = new List();

                                    {

                                        int flag = 0;
                                        foreach (String b in Uniques)
                                        {
                                            if (b == currentline)
                                            {
                                                flag++;
                                            }
                                        }
                                        if (flag == 0)
                                        {
                                            Uniques.Add(currentline);
                                        }
                                    }
