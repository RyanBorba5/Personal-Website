I was able to dig up the code for this project and I'm so happy that I was able to access it because I almost just made this page without the code. Looking back, it's such a mess but it got the job done. Basically we iterate through every single MSB (dark souls map file) and duplicate the enemies in every single one. This project used a repository called SoulsFormats that helped me access specific objects and parts in the game files. This was also my first ever exe I've made which I probably stated already but it still makes me feel accomplished. I actually played this myself which was fun and it's a real interesting experience playing one of the hardest fromsoftware games with multiplied enemies. I wish I had my reaction to getting it to work because I was so so excited.
Click the github icon for a link to the project's source code!
